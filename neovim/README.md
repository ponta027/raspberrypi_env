# neovim


## Local Install

clone  neovim dot file.

> git clone https://github.com/ponta027/dotfiles.git ~/git/dotfiles

> ./setup.sh

> ./install_local.sh

## Docker version
clone  neovim dot file.

> git clone https://github.com/ponta027/dotfiles.git ~/git/dotfiles

## build container

> build.sh ${USER}

## setup neovim config

> ./setup.sh

start nvim container

> ./nvim
